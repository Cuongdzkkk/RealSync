# TASK-FE-POSTING-07: Responsive Mobile

**Module**: Posting Management
**Layer**: Frontend — Vue 3
**Priority**: P2
**Depends on**: TASK-FE-POSTING-01 → 06

---

## Mục tiêu

Tối ưu responsive cho tất cả posting views trên mobile/tablet.

## Checklist

- [ ] Post list: table → card view trên mobile
- [ ] Post detail: stack layout, full-width sections
- [ ] Post form: full-width inputs, collapsible sections
- [ ] Calendar: chuyển sang list view trên mobile
- [ ] Analytics: stack charts vertically
- [ ] Navigation: bottom tabs hoặc hamburger menu

## Breakpoints

| Breakpoint | Target |
|------------|--------|
| ≥1024px | Desktop (default) |
| ≥768px | Tablet |
| <768px | Mobile |

## Verification

- Tất cả views hiển thị đúng trên mobile (375px)
- Tất cả views hiển thị đúng trên tablet (768px)
- Touch interactions hoạt động
- No horizontal scroll
